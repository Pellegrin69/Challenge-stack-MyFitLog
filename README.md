# Challenge-stack-MyFitLog

## Pour récupérer le projet :

- cd là_où_tu_veux_cloner_le_projet/
- git clone https://github.com/Pellegrin69/Challenge-stack-MyFitLog.git
- cd Challenge-stack-MyFitLog/myfitlog/

- yarn dev
